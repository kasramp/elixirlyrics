Elixir Lyrics

Free API to get songs lyrics

This project is maintained by kasramp

Welcome to Elixir Lyrics

Elixir Lyrics is a free REST API (Free songs lyrics API) which allows to simply extract the lyric of a song via a web service call.

Technical details

Elixir Lyrics web service is hosted on Openshift and Heroku. It is developed on Spring Boot, Hibernate frameworks and PostgreSQL. The source code will be publicly published soon. Currently the project is in the beta stage and it might have some glitches due to unexpected reasons.

Getting started

For more details about the web service and how to call it checkout our documentation page.
